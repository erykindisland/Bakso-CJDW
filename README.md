# Bakso Tusuk CJDW - Website Profil Perusahaan

Website profil resmi untuk **Bakso Tusuk CJDW** (Established 2000).

## Fitur
- Desain Premium & Modern (Kuning/Merah)
- Responsive & Mobile Friendly
- Video Proses Produksi dengan Replay Thumbnail
- WhatsApp Integration (Booking & Agen Pemasar)
- Scroll Reveal Animations

## Teknologi
- HTML5
- Vanilla CSS3
- JavaScript (ES6+)

## Cara Menjalankan Lokal
1. Pastikan semua file gambar (`poster_utama.jpg`, etc.) dan video (`video-bakso.mp4`) ada di folder utama.
2. Jalankan local server (misal: Live Server atau `npx serve .`).
3. Buka browser di `localhost`.

## Deployment
Website ini siap di-deploy ke **Cloudflare Pages** melalui GitHub.
